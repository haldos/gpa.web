FChT Vamp Plugin

Descripción

Hemos implementado un plugin que puede utilizarse con el software de análisis de audio llamado Sonic Visualiser. Este plugin implementa el cálculo de la Transformada Fan Chirp aplicado al análisis de música polifónica. Esta técnica, basada en la descomposición de la señal de audio en "chirps" armónicos, da como resultado una representación de la prominencia en altura, llamada F0grama.

Implementación

La implementación se basa en el sistema de plugins de procesamiento de audio llamado VAMP. Este sistema es típicamente utilizado para plugins de análisis y extracción de características en señales de audio.

Instrucciones de instalación

  • Linux
    Utilizar los makefiles incluidos, tanto para 32 como para 64 bits. Por ejemplo, para 32 bits, ejecutar en una terminal:
    makefile -f Makefile.linux
    Luego copiar el archivo compilado al directorio donde se instalan los plugins vamp (en general ~/vamp):
    cp FChT_f0gram.so ~/vamp
