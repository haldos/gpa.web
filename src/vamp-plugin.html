FChT Vamp Plugin para Sonic Visualiser

Descripción

Hemos implementado un plugin para el software de edición de audio llamado Sonic Visualiser. Este plugin implementa el cálculo de la Transformada Fan Chirp.

Implementación

La implementación se basa en el sistema de plugins de proesamiento de audio llamado VAMP. Este sistema es típicamente utilizado para plugins de análisis y extracción de características en señales de audio.
